Cortex AI Functions — Unstructured Text

Built-in SQL functions that classify, summarize, translate, extract sentiment, and filter unstructured text — no model training, no infrastructure, just SQL.

  • AI_CLASSIFY — zero-shot categorization into any set of labels
  • AI_SENTIMENT — detect tone and emotion across multi-language text
  • AI_TRANSLATE — translate between 30+ languages with auto-detection
  • AI_COMPLETE — generate text, summaries, rewrites with any supported LLM
  • AI_FILTER / AI_AGG — semantic WHERE and GROUP BY in natural language

Sample Use Cases

  • Auto-tag support tickets by topic and urgency without training data
  • Summarize thousands of customer reviews into actionable themes
  • Filter regulatory filings for ESG-related disclosures using plain English

Cortex AI Functions — Multimodal

Process audio, images, video, and documents directly in SQL. No external pipelines, no pre-processing — pass files from stages to AI functions and get structured results.

  • AI_TRANSCRIBE — speech-to-text with speaker diarization (31 languages)
  • AI_COMPLETE + TO_FILE() — analyze images, video, and documents with any LLM
  • AI_PARSE_DOCUMENT — extract structured text from PDFs, scans, forms
  • AI_FILTER on images — semantic image filtering with natural language
  • Structured output — response_format => TYPE OBJECT(...) for JSON schema enforcement

Sample Use Cases

  • Transcribe and analyze call recordings to detect fraud patterns
  • Extract product information from marketing videos for cataloging
  • Parse invoices and contracts into structured database rows

Cortex Agents + Snowflake Intelligence

Build autonomous AI agents that reason, select tools, execute actions, and reflect on results. Expose the same capabilities to business users through a no-code conversational interface.

  • Cortex Analyst — natural language to SQL via Semantic Views
  • Cortex Search — RAG over unstructured documents
  • Custom Tools — stored procedures for real-world actions (email, APIs, workflows)
  • CREATE AGENT — combine tools with instructions and guardrails
  • Snowflake Intelligence — no-code UI with charts, artifacts, and verified answers

Sample Use Cases

  • Self-service analytics bot that answers business questions and sends scheduled reports
  • IT help desk agent that searches knowledge base and creates tickets automatically
  • Compliance agent that cross-references policy documents with operational data

Cortex Code

AI-powered coding assistant available both inside Snowsight (worksheets & notebooks) and as a standalone CLI for local development. Understands your schema, writes SQL, manages objects, and deploys code.

  • Snowsight integration — inline AI in SQL worksheets and Python notebooks
  • CLI (CoCo) — terminal agent that reads, writes, edits files and runs commands
  • Schema-aware — auto-discovers databases, tables, views, stages
  • Multi-tool — executes SQL, bash, file operations, web fetch in one conversation
  • Extensible — hooks, MCP servers, custom skills, team mode

Sample Use Cases

  • Generate and debug complex SQL queries from natural language descriptions
  • Scaffold dbt models, Streamlit apps, or stored procedures from scratch
  • Explore unfamiliar databases and generate documentation automatically
